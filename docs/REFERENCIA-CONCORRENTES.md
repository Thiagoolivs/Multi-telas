# Referência de concorrentes (features)

Lista de recursos de uma plataforma de digital signage de referência (OptiSigns),
com o mapeamento para o **nosso** estado atual. Serve de norte para roadmap e pitch.

Legenda: ✅ temos · 🔄 parcial · ⬜ falta

## Onde somos diferentes (nossa aposta)
- **Campanha pronta em 1 clique** (IA cria a tela inteira: copy, CTA, prioridade, tom da marca) — foco em **facilidade**, não em ter mil botões.
- **Player tech-premium** coeso com a marca (não template genérico).
- Multiplataforma no roadmap (mesma campanha → player + Instagram/Facebook).

## Gaps priorizados (candidatos de roadmap)
1. **Proof-of-play / relatórios** — forte para venda corporativa.
2. **PDF/PPT** como fonte de conteúdo — pedido comum.
3. **Reinício/reload automático** — estabilidade das telas (barato).
4. **Retrato de verdade** + **modo quiosque** — amplia casos de uso.
5. **Integrações de ficheiros** (Drive/Dropbox) e **social** (IG/FB).
